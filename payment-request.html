🔗 슬랙 채널 연결

결제정보가 자동으로 올라갈 슬랙 채널의 Incoming Webhook URL을 입력하세요. 이 브라우저에만 저장됩니다.

발급 방법 (최초 1회)
1. api.slack.com/apps → Create New App → From scratch
2. 좌측 Incoming Webhooks → Activate On
3. Add New Webhook to Workspace → 채널 선택 → URL 복사
