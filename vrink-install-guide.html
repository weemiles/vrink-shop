1
키오스크 비즈니스 모드 설정
태블릿을 VRINK 전용 단말기로 고정 · 홈버튼 및 알림바 잠금
1
태블릿 설정 앱 실행 → 디지털 웰빙 및 자녀 보호 → 앱 타이머 비활성화
2
디스플레이 → 화면 자동 꺼짐 → 안 꺼짐 또는 30분 설정 / 자동 밝기 해제 후 최대 밝기 고정
3
VRINK 앱 실행 → 우측 상단 설정 아이콘 탭
4
비즈니스 모드 항목 → 활성화 → 앱이 전체화면으로 고정되며 홈버튼 비활성화됨
5
비즈니스 모드 해제 비밀번호 반드시 기록 보관 (분실 시 리셋 필요)
2
관리자 페이지 — 지점 · 키오스크 · 시스템 설정
admin2.vrink.kr 접속 후 진행
2-1 · 지점 등록
1
좌측 메뉴 지점 관리 클릭
2
우측 상단 지점 추가 버튼 클릭 → 아래 항목 입력 후 저장
항목입력 예시
지점 코드BR-010

주소서울시 ○○구 ○○로
연락처010-0000-0000
운영 상태운영중
오픈 / 마감06:00 ~ 23:00
2-2 · 키오스크 등록
1
좌측 메뉴 지점 관리 → 키오스크 클릭
2
키오스크 추가 버튼 클릭 → 아래 항목 입력 후 저장
3
키오스크 태블릿 앱 설정 → 지점 연결 → 등록한 지점 선택
4
관리자 웹 키오스크 목록에서 온라인 상태 확인
항목입력 예시
키오스크 IDKSK-010
설치 지점위에서 등록한 지점 선택
기기 모델Samsung SM-X216N
시리얼 번호SN-XXXX-XXXX (기기 후면 확인)
IP 주소192.168.1.100 (공유기 관리자 페이지 확인)
