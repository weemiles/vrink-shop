1
키오스크 비즈니스 모드 설정
태블릿을 VRINK 전용 단말기로 고정 · 홈버튼 및 알림바 잠금
1
태블릿 설정 앱 실행 → 디스플레이 → 화면 자동 꺼짐 → 10분 설정
2
디스플레이 → 밝기 최적화 비활성화 → 밝기 슬라이더 최대로 고정
3
VRINK 앱 실행 → 우측 상단 QR코드 버튼 길게 누름
4
관리자 비밀번호 0000 입력 → 관리자 메뉴 진입
5
비즈니스 모드 항목 → 활성화 → 앱이 전체화면으로 고정되며 홈버튼 비활성화됨
6
비즈니스 모드 해제 비밀번호 반드시 기록 보관 (분실 시 리셋 필요)

3
디스펜서 관리자 앱 — 네트워크 · 설정
키오스크 태블릿의 디스펜서 관리자 V1.0 앱 · 운영 / 설정 / 재고 / 네트워크 탭
3-1 · 네트워크 연결 확인
1
디스펜서 관리자 앱 하단 네트워크 탭 선택
2
서버 연결 상태 항목에서 SSE 실시간 연결 표시 확인 — "서버의 실시간 연결되어있습니다" 메시지 녹색으로 표시되어야 함
3
현재 연결된 네트워크 항목에서 SSID와 IP 주소 기록 (예: KT_GIGA_5G_09B3 / 172.30.1.93)
4
네트워크 변경이 필요한 경우 사용 가능한 네트워크 목록에서 해당 SSID 선택 → 새로고침 버튼으로 목록 갱신 가능
5
연결 완료 후 상단 상태가 연결됨(녹색)으로 변경되는지 확인
항목확인 내용
서버 연결 상태SSE 실시간 연결 · 녹색 아이콘 확인
현재 연결 네트워크SSID 및 IP 주소 기록
연결 해제 상태빨간색 "연결 해제" 표시 시 Wi-Fi 재연결 필요
