1
디바이스 등록 및 초기 기기 설정
관리자 페이지에서 QR코드 생성 → 태블릿 초기화 → 앱 자동 설치
1-1 · 관리자 페이지 — QR코드 생성
경로
사이드바 → 시스템 관리 → 디바이스
1
디바이스 등록 버튼 클릭 → 지점 선택 → 생성 클릭
2
화면에 QR코드가 표시되면 절대 닫지 말고 그대로 유지 — 태블릿 등록이 끝날 때까지 PC 화면을 켜두어야 함
⚠ QR코드 화면이 꺼지면 등록이 실패합니다. 태블릿 스캔 완료 전까지 PC 화면을 닫지 마세요.
1-2 · 태블릿 초기화 및 등록
1
태블릿 완전 초기화 (공장 초기화) 진행
2
전원 켜기 → 첫 화면 "환영합니다" 글자가 나오면 해당 부분을 연속으로 빠르게 터치
3
QR 스캐너 화면이 나오면 PC에 띄워둔 QR코드를 인식
4
Wi-Fi 연결 후 태블릿 활성화 진행 — 약관 동의 등은 모두 동의 / 예 선택하며 진행
5
세팅 완료 후 키오스크 앱 자동 설치 및 실행 — 자동 실행이 안 되면 설치된 앱을 직접 눌러서 실행
1-3 · 마무리 — 동기화
1
태블릿 앱 실행 완료 확인 후 관리자 페이지로 돌아옴
2
디바이스 등록 버튼 옆 동기화 버튼 클릭 → 최종 등록 완료
2
키오스크 비즈니스 모드 설정
태블릿을 VRINK 전용 단말기로 고정 · 홈버튼 및 알림바 잠금
1
태블릿 설정 앱 실행 → 디스플레이 → 화면 자동 꺼짐 → 10분 설정
2
디스플레이 → 밝기 최적화 비활성화 → 밝기 슬라이더 최대로 고정
3
VRINK 앱 실행 → 우측 상단 QR코드 버튼 길게 누름
4
관리자 비밀번호 0000 입력 → 관리자 메뉴 진입
5
비즈니스 모드 항목 → 활성화 → 앱이 전체화면으로 고정되며 홈버튼 비활성화됨
6
비즈니스 모드 해제 비밀번호 반드시 기록 보관 (분실 시 리셋 필요)
3
관리자 페이지 — 지점 · 키오스크 · 시스템 설정
admin2.vrink.kr 접속 후 진행

2

디스펜서 관리자 V1.0 앱 설정
태블릿 전용 관리자 앱 · 운영 / 설정 / 재고 / 네트워크 4개 탭 순서대로 진행
앱 진입 방법
1
태블릿에서 VRINK 앱 실행 → 홈 화면 좌측 상단 QR코드 아이콘 길게 누름
2

4-1 · 네트워크 탭 — Wi-Fi · 서버 연결 확인
1
하단 네트워크 탭 선택
2
서버 연결 상태: SSE 실시간 연결 · "서버의 실시간 연결되어있습니다" 메시지 + 녹색 아이콘 확인
3
현재 연결된 네트워크: SSID 및 IP 주소 기록 (예: KT_GIGA_5G_09B3 / 172.30.1.93)
4
Wi-Fi 변경 필요 시 → 사용 가능한 네트워크 목록에서 SSID 선택 (목록 갱신: 새로고침 버튼)
5
연결됨(녹색) 상태 전환 확인. 빨간색 연결 해제 표시 시 Wi-Fi 재연결 필요
